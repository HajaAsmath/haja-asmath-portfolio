giftcard.com
E-gift fulfillment
E-Gift Delivery Pipeline
Java · Spring Boot · RabbitMQ · AWS SES · EKS
  • Legacy system blocked the API thread while SendGrid delivered each gift email synchronously — under load, queues backed up at the API layer and customers waited minutes for their card.
  • Re-architected the producer to publish onto a durable RabbitMQ exchange; horizontally-scaling consumers drain the queue in parallel and dispatch via AWS SES.
  • Failed deliveries are routed to a dead-letter queue for inspection; the consumer ASG scales out on queue depth and back in when drained.
  • End-to-end delivery latency dropped from ~10 minutes to ~20 seconds at the same traffic volume — and burst capacity grew with no producer-side changes.
